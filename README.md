# Schedule Parser

Минималистичный API-сервер на FastAPI для парсинга расписаний моей группы в колледже. Автоматически разворачивается как системная служба (systemd).

## Быстрый старт (Deployment)

Запустите эту команду на чистом сервере (Ubuntu/Debian) для автоматической настройки окружения, установки зависимостей и запуска службы:

```bash
curl -sSL https://raw.githubusercontent.com/qfunky/schedule-parser/main/setup.sh | bash
```

Чтобы получить расписание нужно постучаться на 8000 порт сервера на котором развернута служба. Чтобы расписание отображалось в эпл календарике, нужно добавить подписочный каленьдарь.
